# LLM Cost Benchmark 2026

A 2026 empirical benchmark on which AI workloads can safely run on cheaper models, plus multi-dimensional cost optimisation analysis (model substitution, prompt caching, output capping, batch processing, prompt compression).

**Status:** Pre-build. PRD in progress.

This repo will contain:
- The full benchmark methodology
- All test prompts (publicly inspectable)
- Captured model outputs (publicly inspectable)
- Quality scoring rubrics
- The published findings writeup

A research artefact accompanying [InferOps](https://inferops.org), an AI inference efficiency platform currently in pre-build validation (Phase 0.5).
